# UnionCrest – Real Estate Investment Platform

UnionCrest is a real estate investment platform designed to support property acquisition, investment tracking, and property repositioning strategies.

Website:
https://unioncrestholdings.com

---

## Key Features

• Property listing and management
• Investment tracking
• Real estate acquisition workflow
• Full-stack web application architecture

---

## Technology Stack

Frontend
- TypeScript
- HTML / CSS

Backend
- Node.js
- Server-side APIs

Infrastructure
- Docker containerization
- Cloud-ready architecture
- Database-backed services

---

## Architecture

The project follows a full-stack architecture with separate components for:

• Client UI  
• Backend API services  
• Shared modules  
• Containerized deployment  

---

## Purpose of the Project

UnionCrest was created to support real estate investment workflows and demonstrate a scalable architecture for property investment platforms.
